# m228-final-project

Authors: Esther Wang and Shilpa Chowbey

This repository is the final project for BIOENGR m228 at UCLA. Using BRFSS survey data, it aims to determine predictors for diabetic status based on self-reported responses. This study evaluates the performance of XGBoost and random forest models.

models.ipynb contains the training, validation, and evaluation of both models.
